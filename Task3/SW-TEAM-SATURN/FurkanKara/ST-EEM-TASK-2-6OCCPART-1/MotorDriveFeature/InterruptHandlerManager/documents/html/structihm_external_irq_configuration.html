Detailed Description

Configuration structure for External IRQ management.

This structure provides configuration attributes and associated getter and setter function pointers for managing external interrupts.

The attributes include port, pin, and trigger type configurations. Getter and setter functions allow dynamic access and modification of these attributes during runtime.
