Detailed Description

Updates the motor driver indicator lamps based on the current status.

This function reads the indicator status for brake, direction, and speed, and updates the corresponding lamps (brake, direction, and speed) accordingly. The function also handles the lamp states and updates the indicator PDU (Protocol Data Unit).

The function performs the following actions:

  1. Reads the brake status and updates the brake lamp.
  2. Reads the direction status and updates the forward and reverse direction lamps.
  3. Reads the speed status and updates the corresponding speed lamp.
  4. Constructs the final indicator PDU to represent the state of all lamps and updates it.
