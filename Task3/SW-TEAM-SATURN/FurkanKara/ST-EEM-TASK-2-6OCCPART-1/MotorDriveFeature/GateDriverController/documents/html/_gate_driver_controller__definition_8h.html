Detailed Description

Definition header file for the Gate Driver Controller.

This file contains the definitions, data types, and structure declarations for the Gate Driver Controller. It includes the necessary interfaces and dependencies for managing gate driver operations, such as fault detection, Vgs status, driver control, and over-current protection control.
